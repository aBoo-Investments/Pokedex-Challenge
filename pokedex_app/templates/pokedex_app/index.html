Welcome to the Pokedex Challenge!

This is a Django application to browse and discover Pokemon, powered by the PokeAPI.

Explore the world of Pokemon, view their details, stats, and more.

View Pokemon List

Project Goals

This project aims to demonstrate Python/Django skills by integrating with the PokeAPI to create a functional Pokedex. Key features include:

  • Listing Pokemon
  • Showing details for a selected Pokemon
  • Searching and filtering capabilities
  • Displaying evolution chains

About PokeAPI

The PokeAPI is a comprehensive, open RESTful API for all things Pokemon. It provides data on Pokemon, moves, abilities, types, and much more.

All data used in this application is fetched directly from the PokeAPI.
